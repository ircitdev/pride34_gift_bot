# Быстрый старт Pride34 Gift Bot

## За 5 минут до запуска

### 1. Установите Python зависимости

```bash
pip install -r requirements.txt
```

### 2. Настройте администратора

Ваш admin ID уже добавлен в `.env`:
```
ADMIN_IDS=65876198
```

### 3. Добавьте изображения

**КРИТИЧЕСКИ ВАЖНО:** Разместите эти файлы:

```
pride34_gift_bot/
├── logo.png              ← Ваш логотип (в корне!)
└── images/
    └── welcome.jpg       ← Приветственное изображение
```

### 4. Запустите бота

```bash
python main.py
```

Готово! Бот работает.

## Как протестировать

1. Найдите бота: [@PRIDE34_GIFT_BOT](https://t.me/PRIDE34_GIFT_BOT)
2. Отправьте `/start`
3. Пройдите квиз
4. Загрузите фото
5. Получите результат с вашим логотипом!

## Админ-панель

1. Отправьте `/admin` боту
2. Используйте кнопки для управления

## Важные детали

- **logo.png** накладывается на каждое сгенерированное фото внизу по центру
- Размер логотипа автоматически подстраивается (макс. 30% ширины изображения)
- Если logo.png отсутствует, фото создаётся без логотипа
- База данных SQLite создаётся автоматически при первом запуске

## Структура генерации фото

1. Загруженное фото пользователя
2. ↓ Детекция лица (OpenCV)
3. ↓ Наложение на шаблон/базовый фон
4. ↓ Добавление logo.png внизу по центру
5. → Готовое изображение

## Проблемы?

**Бот не запускается:**
- Проверьте установку зависимостей: `pip install -r requirements.txt`
- Проверьте токен в `.env`

**Не генерируется фото:**
- Убедитесь что есть папка `images/`
- Проверьте что OpenCV установлен: `pip install opencv-python`

**Логотип не накладывается:**
- Проверьте что `logo.png` находится в корне проекта (рядом с `main.py`)
- Проверьте логи на ошибки

## Команды бота

- `/start` - начать квиз
- `/admin` - админ-панель (только для ID: 65876198)

## Файлы которые создаются автоматически

- `bot.db` - база данных SQLite
- `user_photos/*.jpg` - загруженные фото пользователей
- `generated_photos/*_christmas.jpg` - сгенерированные изображения

## Следующие шаги

1. Создайте красивое `welcome.jpg` для приветствия
2. Опционально: создайте шаблоны фигурок в `images/templates/`
3. Настройте дату окончания розыгрыша в `.env`
4. Запустите бота и протестируйте полный flow
